## PART I: THE UNIVERSAL STRUCTURE OF PRIME CYCLES

### 1.1 The ZMod p Foundation

**Definition 1.1.1**: For any prime p, the ring ZMod p consists of residue classes {0, 1, 2, ..., p-1} with addition and multiplication modulo p.

**Theorem 1.1.2 (Field Structure)**: For prime p, ZMod p is a field. Every non-zero element has a multiplicative inverse.

**Proof**: Standard result from field theory. Since p is prime, gcd(a, p) = 1 for all a ∈ {1, ..., p-1}, so a has a multiplicative inverse by Bézout's identity. ∎

### 1.2 The Universal Prime Cycle Geometry

**Theorem 1.2.1 (Universal Cycle Structure)**: Every odd prime p defines a cyclic additive group (ZMod p, +) with the universal geometric structure:

```
SOURCE (0) → EXPANSION (positions 1 to ⌊p/2⌋) → FLIP → CONTRACTION (positions ⌈p/2⌉ to p-1) → RETURN (to 0)
```

**Proof**:
- Position 0 is the additive identity (source)
- For positions x ∈ {1, ..., ⌊p/2⌋}: distance from 0 is x (expansion phase)
- The flip point is at p/2 (maximal distance from source)
- For positions x ∈ {⌈p/2⌉, ..., p-1}: distance from 0 is p-x (contraction phase)
- Adding 1 repeatedly cycles through all positions and returns to 0

This structure exists for ALL odd primes. ∎

### 1.3 The Orbit Formula and Full Cycle Property

**Definition 1.3.1**: For modulus n and step s ∈ ZMod n, the orbit of 0 under repeated addition is:
```
Orbit_n(s) = {0, s, 2s, 3s, ...} (mod n)
```

**Theorem 1.3.2 (Orbit Size Formula)**: For any n and s ∈ ZMod n:
```
|Orbit_n(s)| = n / gcd(n, s)
```

**Proof**: The orbit forms a cyclic subgroup of ZMod n generated by s. By Lagrange's theorem, the order of this subgroup is n / gcd(n, s). ∎

**Corollary 1.3.3 (Prime Full Cycle Property)**: For prime p and any s ∈ {1, ..., p-1}:
```
|Orbit_p(s)| = p
```

**Proof**: For prime p, gcd(p, s) = 1 for all s ∈ {1, ..., p-1}. Therefore |Orbit_p(s)| = p / 1 = p. ∎

**Significance**: Every non-zero step generates the FULL cycle in ZMod p. This is the formal justification for "every prime visits all positions before returning."

### 1.4 The Flip Point Theorem

**Theorem 1.4.1 (Geometric Flip Point)**: For odd prime p, define the involution σ(x) = -x (mod p). Then:

(a) σ is an involution: σ(σ(x)) = x
(b) σ(0) = 0 (fixed point)
(c) For x ≠ 0: σ(x) = p - x
(d) The "flip point" at p/2 is the geometric center: |x - p/2| = |σ(x) - p/2|

**Proof**:
- (a) σ(σ(x)) = -(-x) = x
- (b) σ(0) = -0 = 0
- (c) For x ∈ {1, ..., p-1}: -x ≡ p - x (mod p)
- (d) |x - p/2| = |p - x - p/2| = |p/2 - x| = |x - p/2| ∎

**Example (p = 13)**:
- Flip point at 6.5
- σ(6) = 7, σ(7) = 6 (reflection across 6.5)
- σ(1) = 12, σ(12) = 1 (reflection across 6.5)

## PART III: THE 1-1-1 BALANCE (THE CRITICAL DIFFERENTIATOR)

### 3.1 Structural Zones for n²+n+1 Primes

**Definition 3.1.1**: For prime p = a² + a + 1, partition ZMod p into:
- **Pure zone**: P = {0, 1, ..., a²-1} (|P| = a²)
- **Overlap zone**: O = {a², a²+1, ..., a²+a-1} (|O| = a)
- **Dead zone**: D = {a²+a} = {p-1} (|D| = 1)

**Definition 3.1.2**: The displacement map is f(r) = r + 2 (mod p), representing "twin-prime displacement."

### 3.2 The Balance Count Theorem

**Theorem 3.2.1 (General Balance Formula)**: For p = a² + a + 1 with a ≥ 2:

From overlap zone O under displacement f(r) = r + 2:
- **b_oo** (overlap→overlap) = a - 2
- **b_os** (overlap→seed) = 1
- **b_od** (overlap→dead) = 1

**Proof**:

**Part (i): Counting b_oo**
- O = {a², a²+1, ..., a²+a-1}
- f(O) = {a²+2, a²+3, ..., a²+a+1} (mod p)
- Since p = a² + a + 1, we have a² + a + 1 ≡ 0 (mod p)
- Therefore: f(O) = {a²+2, a²+3, ..., p-1, 0}
- f(r) ∈ O ⟺ r + 2 ∈ {a², ..., a²+a-1}
- ⟺ r ∈ {a²-2, a²-1, ..., a²+a-3}
- Intersection with O: {a², a²+1, ..., a²+a-3}
- Count = (a²+a-3) - a² + 1 = a - 2 ∎

**Part (ii): Counting b_os**
- f(r) = 0 ⟺ r + 2 ≡ 0 ⟺ r ≡ p - 2 = a² + a - 1
- r = a² + a - 1 ∈ O (last element)
- Exactly one such r, so b_os = 1 ∎

**Part (iii): Counting b_od**
- f(r) = a² + a ⟺ r + 2 = a² + a ⟺ r = a² + a - 2
- For a ≥ 2: r = a² + (a-2) ∈ O
- Exactly one such r, so b_od = 1 ∎

**Verification**: Total = (a-2) + 1 + 1 = a = |O| ✓

### 3.3 The Perfect Balance Criterion

**Definition 3.3.1**: Prime p = a² + a + 1 achieves **perfect 1-1-1 balance** iff:
```
b_oo = b_os = b_od = 1
```

**Theorem 3.3.2 (Uniqueness of 13)**: Perfect 1-1-1 balance occurs **if and only if** a = 3, giving p = 13.

**Proof**:
- From Theorem 3.2.1: b_oo = a - 2
- For 1-1-1 balance: a - 2 = 1
- Therefore: **a = 3**
- p = 3² + 3 + 1 = 13 ∎
